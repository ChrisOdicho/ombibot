# OmbiBot

> A bot to ombi with

## Prerequisites

1. Ruby >= 1.9.3
2. Bundler/RubyGems

## Installation

1. Install dependencies

```
$ bundle install
```

## Running

1. Replace environment variables

```
- create a file in the root called `.env`
- copy from `.env.example` and replace with your values
```

2. Start the server

```
$ bundle exec foreman start
```
